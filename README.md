# drawGraph-JSLibrary

JSのCanvas機能を使用したグラフ描画ライブラリです。
IEでは動作しない可能性があります。

## 使用方法

### 棒グラフ

```html
<script>
    // データを二次元配列にて作成
    const GraphData = [
        ["1月",100  ,137 ,156],
        ["2月",200  ,32  ,176],
        ["3月",250  ,167 ,87],
        ["4月",120  ,367 ,257],
        ["5月",300  ,212 ,19],
        ["6月",80   ,78  ,91],
        ["7月",11   ,345 ,187],
        ["8月",0    ,198 ,99],
        ["9月",320  ,0   ,248],
        ["10月",278 ,10  ,378],
        ["11月",156 ,20  ,159],
        ["12月",367 ,30  ,234]
    ];
    // キャンバスのインスタンス生成(id,width,height)
    var MyCan = new drawGraph("MyCanvas",800,500);
    MyCan.drawBarGraph(GraphData);
</script>

<div id="MyCanvas"></div>
```
